Pulsar data portal

The Gravitational Wave Data Centre pulsar data portal provides access to pulsar observations metadata and some data products. Currently, we are hosting observations from the MeerTime project with plans to include more in the future.

The data are recorded with the MeerKAT telescope operated by South African Radio Astronomical Observatory.

Currenlty, we provide public access to all the metadata associated with those observations without the need to log in. Otherwise you can log in to view more data.
